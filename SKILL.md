# 排列5 (Pick5) 预测技能 V1.16.0

10层评分(L1-L10, L5/L7/L8已移除) + 置换检验MI裁剪 + 卡方滑动窗口 + CUSUM断点 + 单路GA + 概率校准 + 基准对比。数据自动同步。

---

## 数据同步

调用 `predict()` 时自动触发。体彩数据API(gameNo=37)获取排列5数据。

## 核心调用

```python
from p5_fusion_complete import Pick5FusionComplete

fusion = Pick5FusionComplete()
result = fusion.predict()
print(result['bets'])  # Top 10推荐
```

或者 CLI：
```bash
python3 p5_lottery_skill.py predict    # 预测下一期
python3 p5_lottery_skill.py info       # 技能信息
```

## 与排列3的关联

排列5的前3位（万/千/百位）与同期排列3开奖号码完全一致。系统利用此相关性：
- **L5排列3关联评分**: 前3位与上期排列3重号越多，评分越高
- **P3和值分布**: 统计P3和值→后2位和值的条件分布，为L2提供参考

## 评分体系 (10层, 置换检验MI动态裁剪)

| 层 | 说明 |
|----|------|
| L1 | 5位位置频率乘积 |
| L2 | 和值+跨度(Gaussian偏差) |
| L3 | 奇偶+012路 |
| L4 | 重复模式(去重数) |
| L5 | 排列3关联(前3位重号) |
| L6 | 条件概率P(后2位\|前3位) |
| L7 | 尾号聚合+AC值 |
| L8 | 跨期差值模式 |
| L9 | 后2位贝叶斯模型 |
| L10 | 位置交互互信息 |

### 附加修正
- **卡方滑动窗口**: 100期窗口, 实时检测偏倚数字, α=0.15软权重(限幅0.6~1.4)
- **CUSUM断点检测**: 双侧CUSUM(k=1.5, h=9), 检测均值结构偏移, 修正L2和值
- **MI置换检验**: 50次标签打乱, 噪声基线校正, 消除窄值域层MI虚高

### 动态权重
置换检验净MI → 归一化 → 保留Top-5层 → 层间冗余校正(皮尔逊相关惩罚) → 二次归一化
